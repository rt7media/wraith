Vidstack

Next-generation video player built with web components

About Vidstack

Vidstack is a next-generation video player built from the ground up using modern web components. It provides a powerful, type-safe API with excellent performance and flexibility for building custom video experiences.

Key Features

  • Built with web components for maximum compatibility
  • Framework agnostic (vanilla JS, React, Vue, Svelte, etc.)
  • TypeScript-first with full type safety
  • Modern architecture with zero legacy code
  • Excellent performance and small bundle size
  • Headless API for complete UI customization
  • HLS, DASH, and YouTube support out of the box
  • Picture-in-Picture and Fullscreen APIs
  • Comprehensive accessibility features
  • Built-in layouts and themes

Best For

Modern web applications, projects using TypeScript, developers wanting complete control over UI/UX, applications requiring custom video experiences, and teams looking for future-proof architecture with web standards.
