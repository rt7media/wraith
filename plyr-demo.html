Plyr

A simple, lightweight, and accessible HTML5 media player

About Plyr

Plyr is a simple, lightweight, accessible HTML5 media player that supports video, audio, YouTube, and Vimeo. It's designed to be easy to use and customize with a beautiful, modern interface out of the box.

Key Features

  • Clean, modern UI with minimal configuration needed
  • Lightweight (only ~10KB gzipped)
  • Fully responsive and mobile-friendly
  • YouTube and Vimeo support built-in
  • Accessible with full keyboard navigation
  • Easy customization with CSS variables
  • Picture-in-Picture support
  • Fullscreen API support
  • Quality selection for adaptive streaming
  • Speed controls and captions

Best For

Projects needing a quick setup with minimal configuration, modern web applications, sites embedding YouTube/Vimeo content, and developers who want a clean UI without extensive customization needs.
