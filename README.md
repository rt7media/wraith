# Wraith - Video Player Demos

A comprehensive collection of open-source HTML5 video player demos for evaluating the best solution for your project.

## ğŸ¬ Featured Players

### 1. [Video.js](https://videojs.com/)
**Industry Standard Player**
- Most widely used open-source HTML5 player
- Extensive plugin ecosystem (100+ plugins)
- HLS and DASH streaming support
- Excellent accessibility (WCAG compliant)
- DRM support (Widevine, PlayReady, FairPlay)
- Best for: Enterprise applications, production environments

### 2. [Plyr](https://plyr.io/)
**Simple & Beautiful**
- Lightweight (~10KB gzipped)
- Clean, modern UI out of the box
- YouTube and Vimeo support built-in
- Easy customization with CSS variables
- Picture-in-Picture support
- Best for: Quick setup, modern web apps, embedded content

### 3. [MediaElement.js](https://www.mediaelementjs.com/)
**Legacy Browser Support**
- Flash and Silverlight fallbacks
- Consistent UI across all browsers
- Support for multiple media formats
- 30+ language translations
- Best for: Legacy browser support, enterprise environments

### 4. [Vidstack](https://www.vidstack.io/)
**Next-Generation Player**
- Built with modern web components
- TypeScript-first with full type safety
- Framework agnostic (React, Vue, Svelte, etc.)
- Headless API for complete customization
- Modern architecture with zero legacy code
- Best for: Modern web apps, TypeScript projects, custom experiences

## ğŸ¯ Choosing the Right Player

### Use Video.js if you need:
- Battle-tested reliability
- Extensive plugin ecosystem
- Advanced streaming features (HLS, DASH, DRM)
- Enterprise-grade support

### Use Plyr if you need:
- Quick and easy setup
- Beautiful UI with minimal configuration
- YouTube/Vimeo embedding
- Lightweight solution

### Use MediaElement.js if you need:
- Support for older browsers (IE11 and below)
- Consistent UI across all platforms
- Flash/Silverlight fallbacks

### Use Vidstack if you need:
- Modern web components architecture
- TypeScript support
- Complete UI customization
- Future-proof solution
