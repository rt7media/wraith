MediaElement.js

HTML5 video and audio player with Flash and Silverlight shims

About MediaElement.js

MediaElement.js is an HTML5 audio and video player that provides a consistent UI across all browsers and includes Flash and Silverlight shims for legacy browser support. It mimics the MediaElement API for older browsers.

Key Features

  • Consistent UI across all browsers and platforms
  • Legacy browser support with Flash/Silverlight fallbacks
  • Support for multiple media formats (MP4, WebM, MP3, etc.)
  • Internationalization with 30+ language translations
  • Skinnable and customizable interface
  • Plugin architecture for extending functionality
  • YouTube, Vimeo, Dailymotion, and SoundCloud support
  • HLS and DASH support via plugins
  • Accessibility features

Best For

Projects requiring legacy browser support (IE11 and older), enterprise environments with diverse user bases, applications needing a consistent player experience across all platforms, and sites serving international audiences.
